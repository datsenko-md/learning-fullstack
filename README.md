# Fullstack-developer course on Hexlet

## Description

Solutions to tasks from the full-stack developer (Node/React) course on [Hexlet](https://hexlet.io/)

[My profile](https://ru.hexlet.io/u/datsenko_md)

## Requirements

* node v18

## Setup

```bash
make install
```

## Run tests

```bash
make test
```
